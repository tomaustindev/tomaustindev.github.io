Module contents

Data package contains Supabase calls, allowing for the front end to access database entries.

data.create_sb_client(auth_header=None) → Client

Creates a Supabase connection. Optionally an authorisation header can be supplied to connect as an already authenticated user.

Parameters:

auth_header – Contains an access token used for authenticating requests.

Returns:

  • Success: A Supabase client/connection used for performing Supabase requests.

  • Failure: [401] BAD_AUTH error code.
